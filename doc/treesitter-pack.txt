==============================================================================
Spec definition                                           *treesitter-pack.spec*

TSPackSpec                                                          *TSPackSpec*

    Fields: ~
        {src}    (string)           Repository address
        {lang?}  (string|string[])  Names of the parsers to install


TSPackOpts                                                          *TSPackOpts*

    Fields: ~
        {force?}     (boolean)  Perform a forced installation
        {generate?}  (boolean)  Generate parser source before building


==============================================================================
API Overview                                               *treesitter-pack.api*

M.add({spec}, {opts?})                                                   *M.add*
    Add parsers to Neovim

    Parameters: ~
        {spec}   (TSPackSpec)  List of parsers marked for installation:
                               * {src} The address is passed as-is to 'git'.
                                 This allows to use a shorter source, like
                                 in |vim.pack-examples|.

                               * {lang} When more than one parser name
                                 is given, the repo is expected to contain
                                 one sub-directory for each name.
                                 These will be considered the new build targets
                                 instead of the repo root.
                                 If omitted, it is inferred from the repository
                                 name by taking the last segment separated by '-'.
        {opts?}  (TSPackOpts)  Other installation settings

    Returns: ~
        (void)

    See: ~
        |TSPackSpec|
        |TSPackOpts|


M.get()                                                                  *M.get*
    Retrieve installed parser names

    Returns: ~
        (string[])


M.del({names})                                                           *M.del*
    Remove parsers from Neovim, deleting their binaries from runtimepath

    Parameters: ~
        {names}  (string[])  Parsers to erase

    Returns: ~
        (void)
